## 🗂️ REPOSITORY ORGANIZATION
The archive is split into two primary branches to distinguish solo survival from legacy gameplay:

### 🧔 CONRAD (Solo Saga)
Located in `/010-Valheim/Chronicles-Of-The-Exile`. These files document the journey of the Exile.
* **Saga I - IV**: From the Meadows to the Mountains.

### 🧒 HEIRS (Grandkids Saga)
Located in `/010-Valheim/020-Heirs/`. These files will document the adventures of the next generation.
* **Saga I**: The Heirs' first steps into the Tenth World. (TBD)

## ✨ AUTOMATION & BRAND CLI
This repository is heavily integrated with the **Brand-CLI** toolset. 

Because this archive grows to thousands of massive transcript files, the automation code has been entirely separated into its own repository to prevent AI assistants from being overwhelmed by token limits.

### [🔗 Go to the Brand-CLI Repository](https://github.com/GrandpaBud/Brand-CLI)
